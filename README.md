# Симулятор активности

Современное портативное приложение на Python, которое имитирует активность пользователя (движения мыши, нажатия клавиш, прокрутка и т.д.), чтобы избежать отслеживания личной активности системами вроде Bitcop.

## Функции

- Выбор целевого окна (Word, PDF, Excel и т.д.)
- Настройка интервала и длительности работы
- Имитация:
  - Печати
  - Прокрутки
  - Движений мыши
- Управление: Запуск, Пауза, Стоп
- Портативный EXE-файл (без установки)

## Установка и запуск

1. Установите [Python 3.12](https://www.python.org/downloads/)
2. Скачайте этот репозиторий как ZIP
3. Распакуйте
4. Запустите `build.bat` — будет создан EXE-файл в папке `dist`
5. Запустите `ActivitySimulator.exe`

## Запуск без сборки

Вы можете запустить приложение напрямую:

```bash
pip install -r requirements.txt
python simulator.py
